\subsection{Detailed Description}
Kernel functions. This namespace contains kernel functions, which evaluate some kernel function $ K(x, y) $ for some arbitrary vectors $ x $ and $ y $ of the same dimension. The single restriction on the function $ K(x, y) $ is that it must satisfy Mercer's condition\-:

\[ \int \int K(x, y) g(x) g(y) dx dy \ge 0 \]

for all square integrable functions $ g(x) $.

The kernels in this namespace all implement the same methods as the \doxyref{Example\-Kernel}{p.}{classmlpack_1_1kernel_1_1ExampleKernel} class. Any additional custom kernels should implement all the methods that class implements; in addition, any method using a kernel should rely on any arbitrary kernel function class having a default constructor and a function


\begin{DoxyCode}
\textcolor{keywordtype}{double} Evaluate(arma::vec&, arma::vec&);
\end{DoxyCode}
