\subsubsection*{template$<$int Power = 2, bool Take\-Root = true$>$class mlpack\-::bound\-::\-H\-Rect\-Bound$<$ Power, Take\-Root $>$}

Hyper-\/rectangle bound for an L-\/metric. 

This should be used in conjunction with the L\-Metric class. Be sure to use the same template parameters for L\-Metric as you do for \doxyref{H\-Rect\-Bound}{p.}{classmlpack_1_1bound_1_1HRectBound} -- otherwise odd results may occur.


\begin{DoxyTemplParams}{Template Parameters}
{\em Power} & The metric to use; use 2 for Euclidean (L2). \\
\hline
{\em Take\-Root} & Whether or not the root should be taken (see L\-Metric documentation). \\
\hline
