\subsection{Detailed Description}
Extremely simple class whose only job is to delete the existing \doxyref{C\-L\-I}{p.}{classmlpack_1_1CLI} object at the end of execution. 

This is meant to allow the user to avoid typing '\doxyref{C\-L\-I\-::\-Destroy()}{p.}{classmlpack_1_1CLI_a4f5b42c89f22052530cace3815bfff32}' at the end of their program. The file also defines a static \doxyref{C\-L\-I\-Deleter}{p.}{classmlpack_1_1util_1_1CLIDeleter} class, which will be initialized at the beginning of the program and deleted at the end. The destructor destroys the \doxyref{C\-L\-I}{p.}{classmlpack_1_1CLI} singleton.
