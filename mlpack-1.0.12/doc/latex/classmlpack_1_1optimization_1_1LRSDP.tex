\subsection{Detailed Description}
\doxyref{L\-R\-S\-D\-P}{p.}{classmlpack_1_1optimization_1_1LRSDP} is the implementation of Monteiro and Burer's formulation of low-\/rank semidefinite programs (L\-R-\/\-S\-D\-P). 

This solver uses the augmented Lagrangian optimizer to solve low-\/rank semidefinite programs.

Create an \doxyref{L\-R\-S\-D\-P}{p.}{classmlpack_1_1optimization_1_1LRSDP} to be optimized. 

The solution will end up being a matrix of size (rank) x (rows).
