\subsection{Detailed Description}
Linear algebra utility functions, generally performed on matrices or vectors. This class is used to update the weight\-Vectors matrix according to the simple update rule as discussed by Rosenblatt\-:

if a vector x has been incorrectly classified by a weight w, then w = w -\/ x and w'= w'+ x

where w' is the weight vector which correctly classifies x.
