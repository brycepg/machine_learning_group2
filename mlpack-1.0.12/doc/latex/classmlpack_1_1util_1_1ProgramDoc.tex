\subsection{Detailed Description}
A static object whose constructor registers program documentation with the \doxyref{C\-L\-I}{p.}{classmlpack_1_1CLI} class. 

This should not be used outside of \doxyref{C\-L\-I}{p.}{classmlpack_1_1CLI} itself, and you should use the \doxyref{P\-R\-O\-G\-R\-A\-M\-\_\-\-I\-N\-F\-O()}{p.}{cli_8hpp_ad781e52fa8bee6ec70cabb646fa16e00} macro to declare these objects.
